RoseAI — Plant Disease Detector

An AI-powered web application that detects diseases in rose plants from a single leaf photo. Built using Google Teachable Machine and TensorFlow.js, it identifies conditions like Black Spot, Downy Mildew, Rose Rust, and Rose Sawfly — providing cause and treatment advice in simple, friendly language. No gardening expertise required.
